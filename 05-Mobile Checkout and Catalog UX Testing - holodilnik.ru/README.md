# 05-Mobile Navigation and Catalog UX Testing - holodilnik.ru

Практический учебный проект по ручному тестированию пользовательского опыта и навигации в мобильной версии интернет-магазина holodilnik.ru.

Проект направлен на проверку удобства навигации в каталоге товаров, доступности категорий и корректности переходов между листингом и карточкой товара на мобильных устройствах.

## Цель проекта

Выявить UX- и навигационные проблемы мобильной версии сайта, которые могут усложнять поиск товаров и возврат пользователя к предыдущим этапам просмотра каталога.

## Объект тестирования

Интернет-магазин holodilnik.ru  
Платформа: Mobile Web  
Устройство: iPhone 14 Pro (iOS 18.6.2)  
Браузер: Яндекс Браузер  
Тип тестирования: Manual QA

## Описание протестированных сценариев

В рамках проекта выполнено тестирование следующих пользовательских сценариев:

- переход из главной страницы в листинг товаров;
- выбор категории товаров из листинга;
- переход из листинга в карточку товара;
- проверка наличия кнопки возврата из карточки товара в категорию;
- возврат пользователя к списку товаров после просмотра карточки;
- проверка прямого доступа к категориям товаров в листинге;
- анализ доступности категории «Часы» без перехода через смежные разделы.

## Типы тестирования

В проекте использовались следующие типы тестирования:

- функциональное тестирование;
- UI/UX-тестирование;
- тестирование навигации;
- негативные сценарии.

## Артефакты проекта

В репозитории представлены:

- тест-кейсы;
- баг-репорты;
- скриншоты с фиксацией навигационных дефектов;
- видеозаписи прохождения пользовательских сценариев;
- результаты выполнения тестов.

## Результаты

В ходе тестирования выявлены следующие проблемы:

- отсутствие кнопки возврата из карточки товара в категорию товаров;
- невозможность прямого доступа к категории товаров «Часы» из основного листинга;
- усложнение навигации и возврата пользователя при просмотре товаров на мобильной версии сайта.

Выявленные дефекты негативно влияют на пользовательский опыт и могут увеличивать количество лишних действий при поиске товаров.
