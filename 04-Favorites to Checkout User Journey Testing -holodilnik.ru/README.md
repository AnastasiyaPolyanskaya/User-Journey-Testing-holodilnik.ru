# 04-Favorites to Checkout User Journey Testing — holodilnik.ru

Практический учебный проект по ручному тестированию пользовательского пути оформления заказа через раздел «Избранное» в интернет-магазине holodilnik.ru.

Проект направлен на проверку корректности прохождения сквозного пользовательского сценария от добавления товара в избранное до этапа оплаты банковской картой, а также на выявление UX- и UI-особенностей интерфейса.

## Цель проекта

Проверить стабильность и корректность пользовательского пути оформления заказа через раздел «Избранное», а также выявить проблемы пользовательского интерфейса, влияющие на удобство взаимодействия с сайтом.

## Объект тестирования

Интернет-магазин holodilnik.ru  
Платформа: Web  
Тип тестирования: Manual QA

## Описание протестированных сценариев

В рамках проекта выполнено тестирование следующих пользовательских сценариев:

- переход из главной страницы в раздел «Избранное»;
- работа раздела «Избранное» при наличии и отсутствии товаров;
- переход из избранного в корзину;
- оформление заказа через корзину;
- авторизация при оформлении заказа;
- выбор способа доставки;
- переход к оплате банковской картой;
- получение кода авторизации и формирование заказа;
- проверка UI-элементов и анимаций на карточках товаров.

## Типы тестирования

В проекте использовались следующие типы тестирования:

- функциональное тестирование;
- E2E-тестирование пользовательского пути;
- UI/UX-тестирование;
- позитивные и негативные сценарии.

## Артефакты проекта

В репозитории представлены:

- тест-кейсы;
- баг-репорты;
- скриншоты с фиксацией выявленных дефектов и UX-особенностей;
- видеозапись прохождения пользовательского пути;
- результаты выполнения тестов.

## Формат результатов

Результаты тестирования оформлены в виде отчёта с указанием:

- пользовательского пути;
- устройства и версии браузера;
- позитивных и негативных моментов;
- ссылок на видео прохождения сценария.

## Результаты

В ходе тестирования выявлены:

- корректная работа пользовательского пути от избранного до оплаты картой;
- быстрая доставка кода авторизации и автоматическое формирование заказа;
- отсутствие анимации переключения изображений на карточках товаров в разделе «Популярные товары» для части товаров;
- отображение числового индикатора в разделе «Избранное» при отсутствии товаров.
