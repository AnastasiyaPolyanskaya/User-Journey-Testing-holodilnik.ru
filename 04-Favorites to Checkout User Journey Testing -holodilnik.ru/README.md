# 04-Favorites and Popular Products UX Testing - holodilnik.ru

Практический учебный проект по ручному тестированию пользовательского опыта и интерфейсных элементов интернет-магазина holodilnik.ru.

Проект направлен на проверку работы раздела «Избранное» и UX-поведения карточек товаров в блоке «Популярные товары» при прохождении пользовательского пути от главной страницы до оформления заказа.

## Цель проекта

Проверить корректность работы функционала «Избранное», а также единообразие и стабильность UI-анимаций карточек товаров в разделе «Популярные товары».

## Объект тестирования

Интернет-магазин holodilnik.ru  
Платформа: Web  
Тип тестирования: Manual QA

## Описание протестированных сценариев

В рамках проекта выполнено тестирование следующих сценариев:

- добавление товара в избранное из блока «Популярные товары»;
- отображение добавленного товара в разделе «Избранное»;
- переход из избранного к дальнейшим этапам пользовательского пути;
- проверка анимации переключения изображений на карточках товаров;
- сравнение поведения анимаций на разных карточках товаров;
- проверка единообразия UI-поведения в блоке «Популярные товары».

## Типы тестирования

В проекте использовались следующие типы тестирования:

- функциональное тестирование;
- UI/UX-тестирование;
- тестирование пользовательского пути;
- негативные сценарии.

## Артефакты проекта

В репозитории представлены:

- тест-кейсы;
- баг-репорты;
- скриншоты с фиксацией выявленных дефектов интерфейса;
- видеозаписи прохождения пользовательских сценариев;
- результаты выполнения тестов.

## Результаты

В ходе тестирования выявлены следующие проблемы:

- добавленный в избранное товар не отображается в разделе «Избранное»;
- анимация переключения изображений работает не на всех карточках товаров;
- отсутствие единообразного UI-поведения карточек в разделе «Популярные товары».

Выявленные дефекты негативно влияют на пользовательский опыт и могут снижать доверие пользователя к интерфейсу сайта.
