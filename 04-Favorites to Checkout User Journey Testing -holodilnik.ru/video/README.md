## Видео

В данном разделе представлены видеозаписи прохождения пользовательских сценариев, в ходе которых были выявлены дефекты в работе раздела «Избранное» и интерфейсных элементов блока «Популярные товары» интернет-магазина holodilnik.ru.

Видео используется для фиксации поведения интерфейса в динамике и подтверждения воспроизводимости обнаруженных дефектов.

## Назначение видео

Видеозаписи позволяют:

- зафиксировать процесс добавления товара в избранное;
- показать отсутствие добавленного товара в разделе «Избранное»;
- продемонстрировать работу анимации карточек товаров при наведении курсора;
- наглядно подтвердить отсутствие анимации на отдельной карточке товара;
- упростить анализ дефектов без повторного воспроизведения сценария.

## Связь с баг-репортами

Видео дополняет следующие баг-репорты:

- **ID: хол4.1 — Товар не добавляется в «Избранное» при нажатии на иконку «сердце»**
- **ID: хол4.2 — Анимация переключения изображений отсутствует на одной из карточек в разделе «Популярные товары»**

Видеоматериалы используются как дополнительное доказательство фактического результата.

## Формат

Видео представлены в формате записи экрана и предназначены для просмотра совместно с тест-кейсами и баг-репортами проекта.
