## Тест-кейсы

В данном разделе представлены тест-кейсы, разработанные для проверки пользовательского опыта и интерфейсных элементов интернет-магазина holodilnik.ru.

Тест-кейсы направлены на проверку работы функционала «Избранное» и корректности UI-анимаций карточек товаров в разделе «Популярные товары».

## Описание тестируемых сценариев

В рамках тест-кейсов проверяются следующие сценарии:

- добавление товара в избранное из блока «Популярные товары»;
- отображение добавленного товара в разделе «Избранное»;
- корректность работы кнопки добавления в избранное;
- проверка анимации переключения изображений на карточках товаров;
- единообразие анимаций при наведении курсора на карточки товаров;
- сравнение поведения анимаций у разных товаров в одном разделе.

## Типы тестирования

Тест-кейсы включают следующие типы тестирования:

- функциональное тестирование;
- UI/UX-тестирование;
- тестирование пользовательского пути;
- негативные сценарии.

## Структура тест-кейсов

Каждый тест-кейс содержит:

- идентификатор тест-кейса;
- описание сценария;
- предусловия;
- шаги воспроизведения;
- тестовые данные;
- ожидаемый результат;
- фактический результат;
- информацию об устройстве и версии браузера;
- статус выполнения (Passed / Failed).

## Формат хранения

Тест-кейсы оформлены в табличном виде и используются для фиксации результатов ручного тестирования интерфейсных элементов и пользовательского поведения.
