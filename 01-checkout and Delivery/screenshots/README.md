## Скриншоты

В данном разделе собраны скриншоты, сделанные в процессе тестирования пользовательских путей интернет-магазина holodilnik.ru.

Скриншоты используются для визуальной фиксации:
- выявленных дефектов;
- некорректной работы пользовательского интерфейса;
- ошибок валидации данных;
- особенностей поведения системы при прохождении сценариев.

## Назначение скриншотов

Скриншоты служат дополнением к тест-кейсам и баг-репортам и позволяют:
- подтвердить наличие дефекта;
- наглядно продемонстрировать фактический результат;
- упростить воспроизведение ошибки;
- зафиксировать текущее состояние интерфейса.

## Структура

Скриншоты могут быть сгруппированы:
- по идентификаторам баг-репортов;
- по этапам пользовательского пути (листинг, доставка, оплата);
- по типу сценария (позитивный / негативный).

## Формат

Скриншоты представлены в графическом формате и сопровождают соответствующие баг-репорты и тест-кейсы.
